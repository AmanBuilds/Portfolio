## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/AmanBuilds/Portfolio.git
```

Go to project folder

```bash
cd Portfolio
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
